## Rules

- UI 打磨会产生大量短周期计划，默认放在本目录，不放在 `docs/plan/` 根目录。
- 控件画板生成的 PNG / JSON / HTML 审查产物继续留在 `.atelier/`，不进入 GitHub。
- 归档计划保留历史上下文，但不作为当前执行入口。
- 新自绘控件进入 `AtelierUI` 共享调用前，仍需用户审查。
